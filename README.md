# Ferrante-Thesis-Repository
This repository contains processing, time-series, plotting, and inversion scripts used for the thesis:

"Multi-Track InSAR Analysis of Afterslip-Dominated Postseismic Deformation Following the 2024 Mw 7.1 Wushi County Earthquake"

The repository includes scripts for GMTSAR-based interferogram processing, deramping, point-based time-series extraction, sampling-based inversion, forward modeling, and figure generation. Large satellite data files and intermediate InSAR products are not included due to file size and data-access limitations.
